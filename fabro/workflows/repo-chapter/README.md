# Repo Chapter Workflow

Reusable Fabro workflow for generating one researched chapter bundle from a
public repository.

Inputs:

- a scaffolded book manifest
- a local checkout of the target repository
- an AutoResearchClaw dossier in the scaffolded research directory

Outputs land in the chapter directory selected by the run config:

- `brief.md`
- `chapter.md`
- `visuals.json`
- `review.md`
